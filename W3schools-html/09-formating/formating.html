HTML Formatting Elements
Formatting elements were designed to display special types of text:
b> Bold text
This text is bold strong> - Important text
This text is important! i> - Italic text
This text is italic em> - Emphasized text
This text is emphasized mark> - Marked text

Do not forget to buy milk today.

small> - Smaller text
This is some smaller text. del> - Deleted text

My favorite color is blue red.

ins> - Inserted text
sub> - Subscript text

This is subscripted text.

sup> - Superscript text

This is superscripted text.

del Defines deleted text

My favorite color is blue red.
